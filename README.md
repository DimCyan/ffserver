## Installation

1. Install dependences via pip
    ```
    pip install -r requirements.txt
    ```

2. Excute `main.py`
    ```
    python -m main
    ```

3. API document: [FFServer API](http://127.0.0.1:8010/docs)

4. Open Browser With URL: [http://127.0.0.1:8010/](http://127.0.0.1:8010/)
